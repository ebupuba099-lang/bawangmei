# DESIGN.md

## 项目与用户画像
- **产品名**：霸王梅（bawangmei）
- **定位**：体彩时时彩"头尾"玩法开奖查询与号码生成工具
- **气质**：传统中式彩票站 + 水墨宣纸 + 梅花国风

## 品牌与视觉方向
- **意象锚点**：水墨山水画底纹，左右红梅枝干点缀，毛笔书法大字标题
- **风格**：国风水墨，典雅复古，纸质质感

## 页面结构

### Header 区域
- 背景：水墨山水 CSS 径向渐变纹理 + 左右两侧 SVG 梅花枝干
- 主标题："霸王梅" — 巨大楷体毛笔书法，深褐色
- 副标题：「43组头尾」— 圆角边框框体

### 核心内容区域
- 期号显示：居中，两侧有细横线装饰
- 状态标签：红橙色胶囊标签，🔥连爆N期 / 数据不稳
- 数据网格：5列×9行，43组 `A xx B` 格式（带空格，无边框，无高亮），全部立即显示无动画

### 底部历史记录区域（最多4期）
- 标题："往期历史" + "真实数据 绝无作假" 红色标语
- 列表项：交替背景色，粉底高亮命中行，白底普通行
- 命中：深红方块 + 白色"中"字
- 未中：灰色空心方框 + 灰色"无"字

### 左侧边栏
- 宽度：192px，半透明白底
- 按钮：复制 | 保存图片分享

## 布局
- **左侧边栏**：192px 宽，半透明白底，2个功能按钮
- **主卡片**：888px 宽，三部分（Header + 内容 + 历史）
- **桌面端优先**

## 动效
- 43 组组合全部一次性显示，无翻牌动画
- 无闪烁/弹跳过度动画

## 设计禁忌
- 禁止使用科技蓝、渐变蓝紫
- 禁止生硬几何装饰
- 禁止过度动画
- 禁止命中组合高亮、禁止组合边框
## 产品定位
"霸王梅" 是中国体彩「排列5」民间选号参考卡，定位为彩民分享/收藏的轻量工具：自动生成 43 组「头尾」组合（形如 4xx6），开奖后对照往期判断是否命中。

## 气质与意象
- **核心意象**：海报风格、纸面印刷、卷宗卡片。低饱和度米黄纸面 + 顶部横条装饰（粉/灰/薄荷绿），区别于"赛博渐变"通用 SaaS 卡片。
- 标题 "霸王梅" 用楷体大写居中，副标题"43组头尾"装在带四角回纹的细线框里。
- 期号"26155期"中间配左右两条细线，下方显示"中/无"状态 + 开奖时间。
- 命中态只用"砖红色"文字色（`#C45A43`）标出当前期命中号，无任何方框/发光/渐变/阴影。

## 视觉策略
- **零装饰**：卡片内部不放花卉/远山/回纹/卷草等 SVG 装饰，画面只保留"色块 + 文字 + 角花小线"。
- 顶部三个 1~6px 高的细横条作为头部装饰（粉、灰、薄荷绿）。
- "43组头尾" 副标题框的四个角落各画一个 10x10 的 SVG 角花（"└/┐/└/┘" 风格直角回纹）。
- 数字采用等宽 sans（PingFang SC / Microsoft YaHei），保证 `4xx6` 字符对齐。
- 命中号用 `#C45A43` 砖红色字，未命中为 `#5A3E2B` 棕字；`xx` 占位与数字同色同字重。

## 字体排版
- 中文标题：系统楷体（`'STKaiti','KaiTi','Songti SC','SimSun',serif`）
- 数字部分：等宽 sans（`'PingFang SC','Hiragino Sans GB','Microsoft YaHei','Source Han Sans SC',sans-serif`）
- 副文字 / 标签：楷体

## 动效与交互
- 加载时数字逐个淡入上浮（30ms stagger，500ms 缓动）
- 按钮 hover 颜色加深；命中号 hover 不变
- 点击"保存图片分享"：iOS / standalone → `navigator.share` 调起系统面板；其他平台 → `<a download>` PNG
- 无任何发光/毛玻璃/渐变/3D 效果

## 页面结构
- **页头**：用户提供的 `header.jpg` 横幅海报（自带标题 + 43组头尾回纹框）
- **主卡片**：
  1. 顶部 3 条装饰横线（粉/灰/薄荷）
  2. "霸王梅"（楷体大字，居中）
  3. "43组头尾"（细线框 + 四角回纹角花，楷体）
  4. 左右两条细线（期号上分隔）
  5. "26155期"（楷体粗体）
  6. "中/无" 状态文字（命中为砖红色，未中为灰色）
  7. 开奖时间（灰小字）
  8. 状态标签（火爆/最近数据不稳，二选一）
  9. 5×9 网格 43 组 `AxxB`（命中用砖红色文字标出，无任何边框/背景）
  10. 4 个按钮：重新生成 / 刷新开奖 / 保存本期结果 / 保存图片分享
  11. 历史开奖（最多 8 期，每行：期号 / 开奖号 / 中无）
- **页脚**：数据来源 + 仅供娱乐参考

## 设计禁忌
- ❌ 卡片内不再画花卉/远山/回纹/卷草等装饰 SVG
- ❌ 渐变背景、阴影发光、毛玻璃
- ❌ 圆角按钮（按钮保持直角或 4px 极小圆角）
- ❌ 蓝色/紫色等冷色调
- ❌ 任何 emoji（🔥 仅作为连爆标签前缀可用，不装饰主内容）
- ❌ "往期"两个字
